# pinned

## about
- from 14/02/26 to 17/02/26, CarnavOwn happened. a CTF competition hosted by Hacking Club that consists in 10 challenges in various categories.
- one of the categories was mobile and i've contributed with one authoral challenge.

## technologies
- app developed on full kotlin
- api developed on full node.js, using express framework
- mongodb used as main database
- nginx for reverse proxy
- docker to orchestrate the containers of node, mongo and nginx
- ec2@aws to host the challenge
